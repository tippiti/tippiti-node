## [1.0.4] – 2026-04-25

### Added

- Initial release of the official TypeScript client.
- Type-safe wrapper around the Tippiti OpenAPI 3.1 specification,
  built on [`openapi-fetch`](https://openapi-ts.dev/openapi-fetch/).
- `createTippitiClient()` factory with token authentication.
- Full type exports for all schemas and operations.
